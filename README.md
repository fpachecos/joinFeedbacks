Projeto que junta todos os feeedback em um unico arquivo

Executar passando os seguintes parametros:

java -jar joinFeedbacks-1.0-SNAPSHOT.jar [caminho para a pasta onde ficam os feedback] [nome do arquivo de saída]

Ex.

java -jar joinFeedbacks-1.0-SNAPSHOT.jar "C:\Stefanini\PJ00212 - BoB\RH\Feedbacks" feedbackConsolidado.xlsx
